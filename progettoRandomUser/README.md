# PROGETTO RANDOMUSER

Questo progetto genera utenti casuali di tutto il mondo, utilizzando l'API randomuser.me

Il sito ha le seguenti funzionalità:

## 1. Sezione filtri
    é possibile filtrare le persone caricate tramite filtri uomo/donna e in base alla nazionalità. Lo slider permette di decidere quante persone generare.

## 2. Modalità singola e modalità griglia
    premendo il button griglia ON/OFF é possibile cambiare il layout degli utenti per rendere piu veloce da visualizzazione di essi.

## 3. Visualizza dettagli utente
    in modalita griglia é possibile visualizzare i dettagli dell'utente spostando il cursose sull'utente interessato. Con la griglia disattivata i dettagli appaiono di default a schermo.

## 4. Frecce per la navigazione
    in entrambe le modalità appaiono 2 button con l'icona della freccia per cambiare utente, i button si disabilitano se uno raggiunge la fine o si trova all'inizio dell'elenco.
